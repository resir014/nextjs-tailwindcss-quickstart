# slatejs-playground

> Playing around with SlateJS!

## Developing

A nodejs >= 14.0.0 setup with [yarn](https://yarnpkg.com/) is recommended.

```bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn dev

# build for production
yarn build
```
